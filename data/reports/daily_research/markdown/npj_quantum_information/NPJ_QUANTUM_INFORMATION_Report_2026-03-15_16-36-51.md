<summary>摘要翻译</summary>

> 摘要在以泄漏错误为主的物理系统中，擦除错误是主要的错误类型。虽然使用稳定子码的量子纠错（QEC）可以对抗擦除错误，但何种构造能达到容量性能仍是未知的。若此类编码存在，人们也期望能在码长上具有线性运行时间的译码器。本文提出了在最大似然译码（MLD）下达到擦除容量的量子码，尽管MLD需要码长的三次方运行时间。对于QEC而言，使用具有最短可能运行时间的精确译码器，将能在等待译码器决策的同时最小化量子信息的劣化。为此，我们提出了置信传播（BP）译码器，其以线性时间运行，并利用稳定子码中的错误简并性，为包括自行车码、乘积码和拓扑码在内的广泛编码类别实现了容量或接近容量的性能。此外，我们还探索了所提出的BP译码器处理混合擦除与去极化错误的潜力，以及通过与置换不变码级联来处理局部删除错误的潜力。

</details>

<summary>摘要 (Abstract)</summary>

> Abstract Erasures are the primary type of errors in physical systems dominated by leakage errors. While quantum error correction (QEC) using stabilizer codes can combat erasure errors, it remains unknown which constructions achieve capacity performance. If such codes exist, decoders with linear runtime in the code length are also desired. In this paper, we present erasure capacity-achieving quantum codes under maximum-likelihood decoding (MLD), though MLD requires cubic runtime in the code length. For QEC, using an accurate decoder with the shortest possible runtime will minimize the degradation of quantum information while awaiting the decoder’s decision. To address this, we propose belief propagation (BP) decoders that run in linear time and exploit error degeneracy in stabilizer codes, achieving capacity or near-capacity performance for a broad class of codes, including bicycle codes, product codes, and topological codes. We furthermore explore the potential of our BP decoders to handle mixed erasure and depolarizing errors, and also local deletion errors via concatenation with permutation invariant codes.

</details>
